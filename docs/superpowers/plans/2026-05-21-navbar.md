# Navbar Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Responsive navbar mit Logo immer links, Desktop-Links mittig, Hamburger rechts (mobile), DaisyUI details/summary Dropdowns für CX Solutions und IN Solutions.

**Architecture:** Drei Astro-Komponenten — Navbar-Shell verwaltet die Layout-Zonen, CXSolution und INSolution liefern Submenu-Items die in Desktop- und Mobile-Menü gleichermaßen wiederverwendet werden. Kein JavaScript nötig; DaisyUI details/summary übernimmt das Toggle-Verhalten.

**Tech Stack:** Astro 6, DaisyUI 5, Tailwind 4 via @tailwindcss/vite
